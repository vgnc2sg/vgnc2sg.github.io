Schnell mal Magento 2 auf einem Windows-System aufsetzen - Stichpunkte

Stell dir vor du bist auf einem Magento Hackathon und nix mit Magento auf deinem Rechner.

Schritt für Schritt

  1. Einen lokalen Server einrichten. (mehr)
  2. Datenbank vorbereiten. (mehr)
  3. Composer Herunterladen und installieren. (mehr)
  4. Magento herunterladen entpacken und Co. (mehr)

Einen lokalen Server einrichten.

Ich hab' bei mir auf dem Rechner XAMPP installiert. und einen virtuellen Host für meine Magento 2 Seite eingerichtet.

C:\xampp\apache\conf\extra\httpd-vhosts.conf 

   <VirtualHost *>
      DocumentRoot "C:/xampp/htdocs/magento2.local"
      ServerName magento2.local
      DirectoryIndex index.php
      <Directory "C:/xampp/htdocs/magento2.local">
        AllowOverride All
        Allow from All
      </Directory>
    </VirtualHost>

im besten Fall läuft jetzt ein lokaler Server mit MySQL und Co.

Datenbank vorbereiten.

Über phpMyAdmin eine Datenbank für Magento 2 erstellen (utf8_general_ci).

Composer Herunterladen und installieren.

Composer Download

Magento herunterladen entpacken und Co.

  • Magento 2 bei GitHub herunterladen
  • entpacken in: C:/xampp/htdocs/magento2.local
  • in den Ordner wechseln, Eingabeaufforderung öffnen, composer install ausführen.
